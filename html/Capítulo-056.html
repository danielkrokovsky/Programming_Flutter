Part 2
Doing More with Flutter

Flutter is expandable and flexible. We’ve already seen how it works, so now we’ll look at how Flutter’s many features can be used to actually build useful apps.

In the chapters in this part you’ll learn how to make HTTP requests, how to test and handle exceptions in Flutter apps, and then how to use Firebase to create a chat app.

The appendices will be about Dart syntax, Cupertino (iOS-like) widgets, pubspec.yaml, and how to change app name and launcher icon.
