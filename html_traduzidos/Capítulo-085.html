Comentários no Dart

Comentários são seções de código que não serão avaliadas pelo compilador e não serão executadas. Assim como em C, Java, JavaScript e linguagens semelhantes, os comentários são criados de duas maneiras:

  • Comentários de linha única são criados adicionando // antes do comentário.
  • Comentários de várias linhas são criados adicionando / * antes do comentário e * / depois dele.

Por exemplo:

​  // isto é um comentário
​  int a = 3; // este é também um comentário
​  / * Este é também
​  um longo, longo
​  comentário * /

onde int a = 3; é um código normal (não um comentário) e será avaliado pelo compilador e executado.
