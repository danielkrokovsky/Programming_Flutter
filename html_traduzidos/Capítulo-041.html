Faça o aplicativo parecer e funcionar direito com expandido

Atualmente, o aplicativo não pareceria certo se você o executasse em um dispositivo.

Para fazer com que os botões tenham a forma e o tamanho corretos, precisamos envolver as linhas , as colunas e os próprios botões em Expandido .

Expanda os elementos para dividir o espaço dentro da coluna principal

Expandiremos alguns elementos para torná-los exatamente tão grandes quanto queremos que sejam.

Neste caso, a Coluna principal será dividida da seguinte forma:

  • 2/7 do espaço serão ocupados pela tela da calculadora.
  • 1/7 do espaço será ocupado pela linha de exclusão.
  • Os 4/7 restantes do espaço (quatro vezes mais que a linha de exclusão) serão ocupados pelos números e operadores, de modo que todos os botões (incluindo os botões de exclusão) tenham a mesma altura.

Como foi explicado em preencher o espaço disponível na exibição usando expandido , o Expanded Widget faz isto realmente simples, exigindo-lhe para substituir a tela da calculadora mostrado no código com este:

layout / calculator_part2 / lib / main.dart
​  Expandido (
​  flex: 2,
​  criança: Cartão (
​  cor: Colors.lightGreen [50],
​  filho: Preenchimento (
​  preenchimento: EdgeInsets.all (15.0),
​  filho: Texto (
​  _str,
​  textScaleFactor: 2.0,
​  ),
​  ),
​  ),
​  ),

Da mesma forma, a primeira fileira widget (a que contém os botões de exclusão, mostrados no código ) terão de ser substituídas por:

​  Expandido (
​  flex: 1,
​  filho: Row (
​  ...
​  )
​  )

e a uma contendo as teclas numéricas (mostrado no código ) irá de forma semelhante ter de ser substituído com:

​  Expandido (
​  flex: 4,
​  filho: Row (
​  ...
​  )
​  )

Expanda os elementos para dividir o espaço dentro da linha de exclusão

Três quartos do espaço serão ocupados pelo botão excluir tudo e um quarto pelo botão excluir um .

Isso significa que esses dois FlatButton s terão que ser expandidos substituindo-os por FlatButton s envolvidos em dois widgets expandidos com valores flex de, respectivamente, 3 e 1:

layout / calculator_part2 / lib / main.dart
​  Expandido (
​  flex: 3,
​  filho: FlatButton (
​  filho: Texto (
​  'C' ,
​  style: TextStyle (color: Colors.white),
​  ),
​  onPressionado: () {deleteAll ();},
​  cor: Colors.black54,
​  ),
​  ),
​  Expandido (
​  flex: 1,
​  filho: FlatButton (
​  filho: Texto (
​  '<-' ,
​  style: TextStyle (color: Colors.white)
​  ),
​  onPressionado: () {deleteOne ();},
​  cor: Colors.black87,
​  ),
​  ),

Expanda os elementos para dividir o espaço dentro do número e da grade do operador

Da mesma forma, você precisará expandir a coluna dos números com um flex de 3 e a coluna dos operadores com um flex de 1 para fazer todos os botões do mesmo tamanho.

Crie ExpandedRow e ExpandedButton para o resto dos botões

Os botões de número e operador devem ser todos do mesmo tamanho, então eles devem ser agrupados em widgets expandidos com o mesmo flex que, para simplificar, definiremos com o valor 1.

O mesmo é verdade para as Row s de botões numéricos: todos eles precisam ser expandidos com o mesmo flex .

Para evitar a repetição e acelerar a substituição de muitos widgets, construiremos classes FlatButton e Row expandidas , que serão chamadas de ExpandedButton e ExpandedRow .

Eles precisam pegar todos os argumentos e opções que usamos atualmente e construir os widgets correspondentes embalados em Expandido .

Eles são definidos, de forma muito simples, acima do principal , da seguinte forma:

layout / calculator_part2 / lib / main.dart
​  classe ExpandedButton estende StatelessWidget {
​ 
​  ExpandedButton ({ este .onPressed, este .child, este .Color});
​ 
​  última criança Widget;
​  definitiva VoidCallback onPressed;
​  última cor cores;
​ 
​  @sobrepor
​  Construção de widget (contexto BuildContext) =>
​  Expandido (
​  flex: 1,
​  filho: FlatButton (
​  onPressed: onPressed,
​  criança: criança,
​  cor: cor,
​  ),
​  );
​  }
​ 
​  classe ExpandedRow estende StatelessWidget {
​ 
​  ExpandedRow ({ este .children, este .crossAxisAlignment});
​ 
​  finais List <Widget> crianças;
​  definitiva CrossAxisAlignment crossAxisAlignment;
​ 
​  @sobrepor
​  Construção de widget (contexto BuildContext) =>
​  Expandido (
​  flex: 1,
​  filho: Row (
​  crianças crianças,
​  crossAxisAlignment: crossAxisAlignment,
​  ),
​  );
​  }

Depois de defini-los, simplesmente substitua todos os widgets Row do botão do operador por widgets ExpandedRow e todos os FlatButton s de número do operador e o layout da calculadora devem estar completos.
