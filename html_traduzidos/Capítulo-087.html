Funções

O Dart oferece suporte a funções de nível superior, o que significa que você pode definir funções fora de qualquer definição de classe que pode ser chamada de qualquer lugar no código.

Uma função Dart é definida da seguinte maneira:

​  returnType functionName (parâmetros) {
​  blockOfInstructions;
​  Retornar returnValue;
​  }

Onde:

  • returnType é o tipo de dados, consulte a seção conversão entre tipos Native Java / Apple e Dados Dart , a função retornará.

  • Os parâmetros, explicados no próximo parágrafo, que serão os dados que a função usará para fazer o que precisa. Os parâmetros são variáveis ​​e seu valor é definido usando argumentos de chamada de função. Se você não estiver familiarizado com funções e chamadas de função, o exemplo a seguir foi projetado para fornecer um exemplo real e funcional de algumas funções que tiram proveito de outras funções.

  • Nas chaves é onde você adiciona o bloco de instruções (uma ou mais linhas) que, se a função tiver qualquer tipo de retorno diferente de void , inclui uma instrução de retorno , que é usada para retornar um valor para o código de chamada.

Por exemplo, a seguinte função é muito simples:

​  duplo quadrado ( duplo n) {
​  regresso n * n;
​  }

Parâmetros de Função

Até agora, usamos apenas parâmetros posicionais: você os define como variáveis, uma após a outra, e define os argumentos da mesma forma, seguindo a mesma ordem.

Agora, porém, você precisa saber sobre os parâmetros nomeados , que permitem que a função de chamada defina os argumentos em uma ordem diferente (ou mesmo para apenas alguns dos parâmetros) especificando um nome para cada parâmetro. Se você envolver os parâmetros entre chaves, você os tornará nomeados e opcionais , como no exemplo a seguir:

​  duplo calculateSunPlanetAttraction ({})

A flecha

Se a função for simplesmente composta de uma instrução, você pode usar a notação de seta, de modo que:

​  int quadrado ( int n) {
​  regresso n * n;
​  }

torna-se:

​  int quadrado ( int n) => n * n;

geralmente escrito, especialmente quando a expressão é mais longa, como:

​  int quadrado ( int n) =>
​  n * n;

Funções anônimas e fechamentos

Fechamentos são o que usamos para definir retornos de chamada. Eles são funções definidas como variáveis ou, mais especificamente, as variáveis do tipo de função ou Typedef e chamadas de retorno .
