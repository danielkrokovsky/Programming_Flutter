O StreamBuilder

O StreamBuilder funciona como um FutureBuilder , mas atualiza seu conteúdo em tempo real para corresponder ao conteúdo de um determinado fluxo , em vez de simplesmente exibir o resultado de um determinado futuro como o FutureBuilder faz:

​  StreamBuilder (
​  stream: myStream (),
​  construtor: (contexto, instantâneo) {
​  se (snapshot.hasData!) return CircularProgressIndicator ();
​  retorno MyWidget (
​  data: snapshot.data
​  );
​  }
​  )
