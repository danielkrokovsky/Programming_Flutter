Typedef e Callbacks

typedef é o que é usado para criar tipos de retorno de chamada como VoidCallback , que é um tipo de retorno de chamada nulo para funções que não requerem argumentos. Um retorno de chamada é um tipo de encerramento fornecido como um argumento para uma função a ser chamada quando algumas condições são atendidas. Um fechamento é uma função que é definida como uma variável. O tipo VoidCallback é um tipo especial de Função que não aceita argumentos e tem um tipo de retorno void e pode ser definido da seguinte maneira usando typedef :

​  typedef VoidCallback = vazio Função ();

Se você quiser definir um retorno de chamada de um tipo diferente que leva algum tipo de argumento, você escreveria o seguinte:

​  typedef IntFunctionCallback = int Função ( int n);

ou a seguinte definição matemática muito comum de uma função real de uma variável real:

​  typedef RealFunctionOfARealVariable = dupla função ( duplo n);

As definições podem ser independentes de tipo (este tipo de sintaxe pode ser usada com qualquer coisa no Dart):

​  typedef OneArgumentCallback <T> = Função T (T a);

Depois de definir esse tipo ou apenas usar o tipo genérico Function , podemos criar funções da mesma forma que criamos variáveis. Por exemplo, se nós queria recriar em Dart equivalente a f (x) = e x função exponencial (onde e é o número de Euler, mais conhecido em alguns países como o número de Napier), gostaríamos de escrever o seguinte, usando pow e e de dardo : matemática :

​  RealFunctionOfARealVariable f = (x) => pow (e, x);
​ 
​ 
​  Assert (f (2) == POW (e, 2));
​  Assert (f (1)> 2,71 && f (1) <2,72);
