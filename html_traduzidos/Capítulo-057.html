Rede e armazenamento I / O e navegação

Escrever aplicativos que estão isolados no próprio telefone do usuário é bom para alguns aplicativos, mas a grande maioria dos aplicativos precisa interagir com uma API da web ou back end, que é o que faremos neste capítulo.

A biblioteca padrão do Flutter não inclui um componente de rede padrão: embora você possa realmente usar a rede para buscar imagens, você não pode fazer solicitações HTTP padrão usando apenas a biblioteca padrão do Flutter / Dart.

É por isso que vamos falar sobre isso agora: Flutter tem um pacote desenvolvido pelo Google no Pub chamado http , que nos permite fazer solicitações HTTP e conectar nosso aplicativo a uma API Web.
