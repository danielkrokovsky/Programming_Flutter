Declarações de afirmação

Em algum lugar no meio do caso de uso de teste e do caso de uso de exceções, você encontrará declarações assert , que são uma forma de o desenvolvedor definir as condições que devem ser atendidas durante a execução do aplicativo ou para ter uma maneira eficaz de testar o resultado de uma determinada condição.

Uma instrução assert pode ser adicionada em qualquer lugar dentro do código de um aplicativo e encerrará a execução do aplicativo lançando uma exceção AssertionError quando uma condição não retornar verdadeira :

​  int a = 3;
​  Assert (um == 3);

Essas duas linhas de código não afetarão a execução do aplicativo e a sintaxe semelhante é frequentemente usada na documentação ao demonstrar os recursos de uma linguagem de programação em exemplos de código, incluindo o Dart Language Tour oficial do Google. [45]

Por outro lado, ter um código como este:

​  int a = (6 + 9/3) / 3;
​  assert (! a = 3);

encerrará a execução do aplicativo lançando uma exceção durante a depuração . As instruções assert não têm nenhum efeito no código de produção e são consideradas apenas durante a depuração do código.

É possível incluir uma string para descrever a asserção, passando-a como o segundo argumento nomeado para afirmar :

​  assert (condição, "A condição não foi cumprida" );
