Personagens e Strings

Em algumas linguagens, existem tipos de char para caracteres únicos e matrizes de char s ou tipos separados para strings. No Dart, o tipo String é o único tipo de caractere / string e pode conter um ou mais caracteres. Por esse motivo, ao contrário de outras linguagens, você pode usar aspas simples ou duplas para delimitar o que você atribui à string (chamado de literal de string ):

​  Cordas saudação = "Hi!" ;

funcionará tão bem quanto:

​  var saudação = 'Oi!' ;

Como em muitas outras linguagens modernas, duas strings podem ser concatenadas usando o mesmo operador + que você usa para a adição entre os números:

​  var firstGreeting = "Hi" + "!" ;
​  assert (firstGreeting == 'Oi!' );

O que é afirmado ? É algo que você vai se acostumar a ver lendo este apêndice: seu efeito no código compilado é que, em uma situação de teste / depuração (não faz nada no código de produção), ele termina a execução se a condição entre parênteses não for verdadeira, mas é amplamente usado em tutoriais e documentação de linguagem de programação para explicar os recursos da linguagem. Sempre que você vir um neste apêndice, deve esperar que a condição entre parênteses seja sempre verdadeira.

A barra invertida

Assim como em muitos outros idiomas, a barra invertida pode ser usada para escapar alguns caracteres ou inserir caracteres especiais.

Por exemplo, se você quiser colocar aspas dentro de uma string, há duas maneiras de fazer isso:

  • Você usa diferentes tipos de aspas para incluir a string (por exemplo 'Ele disse: "Oi!"' ).

  • Você escapa das aspas (por exemplo, "Ele disse: \" Oi! \ "" ).

Para a parte dos caracteres especiais, isso será familiar especialmente para desenvolvedores C; o caractere \ n pode ser usado para ir para uma nova linha:

​  Texto ( "primeira linha \ n Segunda linha" )

irá renderizar a segunda linha abaixo da primeira linha .

Obviamente, a barra invertida em si deve ser escapada para ser usada como parte de uma string:

​  Texto ( " \\ é a barra invertida" )

exibirá \ é o caractere de barra invertida .

Interpolação de String

A interpolação de string permite inserir o valor de uma ou mais variáveis ​​em uma string. Vamos primeiro explorar como as coisas seriam feitas sem a interpolação de strings.

Por exemplo, se você quiser exibir o valor de um int com um Texto , não pode simplesmente escrever:

​  int número;
​ 
​  // (...)
​ 
​  Texto (número)

porque Text espera uma String . Isso significa que você precisa mudar para:

​  Texto (number.toString ())

E se você quiser exibir uma string e o número? Então você precisaria escrever algo assim, usando concatenação de string:

​  "Você bateu no botão" + number.toString () + "times"

O que a interpolação de string permite que você faça é escrever isso:

​  "Você bateu o botão de número $ vezes"

Ao prefixar o nome de uma variável com $, você pode inserir seu valor na string. Se você precisa acessar um elemento de uma Lista , o membro de uma classe, ou escrever qualquer tipo de expressão, você precisa colocar a instrução entre chaves. Se você tivesse uma lista como a seguinte:

​  Números da lista = [1, 2, 3];

você usaria interpolação de string como esta:

​  "Existem $ {numbers.length} e segundo elemento é o número $ {números [1]} "

Multi-Line Strings e Raw Strings

Como em muitas outras línguas, você pode fazer strings de várias linhas usando aspas triplas e pode fazer strings brutas prefixando as aspas iniciais com a letra “r”.

Uma string de várias linhas permite que você gire:

​  "Primeira linha \ n Segunda linha" ;

ou isto:

​  "Primeira linha \ n " +
​  "Segunda linha"

nisso:

​  """
​  primeira linha
​  segunda linha
​  """

Uma string bruta trata quaisquer caracteres especiais dentro dela ( $ , \ e assim por diante) como qualquer outro caractere, sem dar a ele qualquer significado especial que você possa escrever apenas:

​  r "\ é a barra invertida"

Variáveis ​​imutáveis: final e const

Se a variável não precisar variar, existem duas palavras-chave que podem garantir que o compilador saiba o que você deseja: final e const :

  • O valor de uma variável final não pode ser alterado: você a inicializa com um valor e esse valor é armazenado lá para sempre, isso é necessário para todas as propriedades do widget no Flutter porque os próprios widgets são imutáveis.

  • const é usado para criar constantes de tempo de compilação: o compilador avaliará o valor da constante quando você construir seu aplicativo, portanto, você não pode ter argumentos const .

Use essas duas palavras-chave substituindo a palavra var por elas:

ou prefixando o tipo de dados variável com um deles:

​  última Cordas saudação = "Hi!" ;

Listas e literais de coleção

Uma Lista é o que comumente se chama de array: uma coleção ordenada de dados, cujos elementos são indexados com os valores de 0 a comprimento-1, onde o comprimento de uma Lista é o número de elementos que ela contém. Uma lista pode ser inicializada com um literal de lista, que é uma lista de valores separados por vírgulas entre colchetes:

Adicionando e removendo itens de listas

Um item pode ser adicionado ao final de uma lista usando List.add :

​  var l = [1, 2, 3];
​ 
​  1. adicionar (4);
​ 
​  Assert (l [3] == 4);

Um item pode ser removido de duas maneiras principais (existem outras, sinta-se à vontade para verificar todos os métodos de lista na referência de linguagem oficial do DART [55] ). Um deles é passar o valor que você deseja remover com List.remove (que removerá apenas a primeira ocorrência):

​  var l = [ "teste" , "teste" , "Test" ];

​  l.remove ( "Ensaio" );
​ 
​  Assert (l [0] == "teste" );

A outra é passar o índice no qual você deseja remover um elemento para List.removeAt :

​  var l = [5, 6, 7];
​ 
​  l.removeAt (1);
​ 
​  Assert (l [1] == 7);

map e forEach

Particularmente amado por alguns por seu estilo funcional, o Iterable do mapa e forEach métodos são alternativas para o para em loop.

forEach é o mais semelhante ao loop for , pois é usado para executar uma operação em (ou para) cada um dos elementos de uma lista. O encerramento de retorno de chamada que você usa é do tipo void , portanto, você não pode retornar nada. Aqui está um exemplo, mostrando como imprimir cada elemento de uma lista:

map é diferente: ele é usado para criar um Iterable a partir da chamada de uma função em cada elemento da Lista . Aqui está um exemplo:

iter é um Iterável , não uma Lista adequada . Se quiser acessar os elementos diretamente da mesma forma que faz com uma lista, você precisa usar o método toList :

O sublinhado

Você deve ter visto em muitos exemplos de código que prefixamos alguns nomes de variáveis ​​ou classes com um sublinhado ( _ ). É usado para torná-los uma biblioteca privada .

Estamos tornando a biblioteca de classes MyWidgetState privada , o que significa que ela só é acessível de dentro do pacote atual: se você estiver trabalhando em uma biblioteca e quiser tornar uma classe ou função de nível superior inacessível para os usuários dessa biblioteca, use o sublinhado Por aqui.

Usar o sublinhado para os membros da classe é usado para torná-lo inacessível de fora de sua definição de classe, como no exemplo a seguir:

o que é obviamente muito mais complicado do que precisa ser ( _setCounterTo adiciona complicações desnecessárias) para mostrar como diferentes tipos de métodos podem ser escolhidos para serem definidos como privados: se estamos criando uma classe de contador, queremos ser capazes de obtém o valor atual do contador, mas não precisamos alterá-lo para qualquer valor diferente de incrementado em 1, portanto, só damos acesso a um método que faz exatamente isso e a outro que retorna o valor do contador.

Números: o tipo num e suas subclasses

A classe num é uma classe numérica genérica e suas subclasses são int para inteiros e double para números de ponto flutuante. Não há flutuação em Dart; todos os números de ponto flutuante são double s de 64 bits .

Seguindo a notação matemática usada no Reino Unido e nos Estados Unidos, o separador decimal é um ponto ( . ). Você pode especificar potências de 10 usando e :

​  var a = 5.2e2;
​ 
​  Assert (a == 520);

os números duplos podem ser arredondados usando round (que arredonda de 0 com a parte decimal começando em 0,5), arredondados para baixo usando floor ou arredondados para cima usando ceil :

​  var a = 5.2;
​ 
​  Assert (a.round () == 5);
​  Assert (a.floor () == 5);
​  Assert (a.ceil () == 6);
​ 
​  var b = 5,5;
​ 
​  Assert (a.round () == 6);

Um inteiro hexadecimal é obtido prefixando-o com 0x :

​  Assert (0x7E3 == 2019);

Listas de bytes e mais: a biblioteca dart: typed_data

Ao ler dados binários, usar uma Lista de int s de 64 bits é extremamente ineficiente se tudo o que cada um desses inteiros está fazendo é claramente menos eficiente do que o ideal. É por isso que o dart: typed_data contém a classe Uint8List , que funciona exatamente como a List regular , mas com inteiros de 8 bits.

Fundição de Tipo

A fundição de tipos no Dart é feita de uma das duas maneiras a seguir.

Você pode usar um como expressão de lançar um valor a um de seus subtipos:

​  Num a = 5;
​ 
​  var a_int = um como int ;
​ 
​  Assert (a_int é int );

ou use cast para converter todos os valores de List para um tipo especificado:

​  List < num > l = [1, 2, 3];
​ 
​  var l_int = l.cast < int > ();
​ 
​  Assert (l_int [0] é int );
