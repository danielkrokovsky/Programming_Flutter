Vamos mostrar uma tela para a calculadora

Obviamente, a calculatorScreen no topo da coluna é apenas um espaço reservado; precisamos substituí-lo por uma tela real de trabalho que será composta do seguinte: um invólucro e um cartão e, dentro dele, um texto preenchido exibindo uma string , que representa a expressão ou resultado a ser exibido.

Mas, antes de adicionar isso, precisamos declarar essa String dentro da declaração _CalculatorHomePageState , acima da palavra-chave @override que precede a declaração do método de construção:

layout / calculator_part1 / lib / main.dart
​  Cordas _str = "0" ;

Depois de ter feito isso, substitua calculatorScreen (do código ) com:

layout / calculator_part1 / lib / main.dart
​  Cartão(
​  cor: Colors.lightGreen [50],
​  filho: Preenchimento (
​  preenchimento: EdgeInsets.all (15.0),
​  filho: Texto (
​  _str,
​  textScaleFactor: 2.0,
​  ),
​  ),
​  ),

Onde Colors.lightGreen [50] é um verde claro de intensidade muito baixa.

A intensidade da cor (50, neste caso) é especificada em uma escala onde 500 é a intensidade normal (como usar Colors.lightGreen ) e números mais altos são cores de intensidade mais alta, geralmente usadas para fins de cores de destaque .
