Layout horizontal: a linha

A linha apresenta os widgets lado a lado e é muito semelhante a uma coluna , com a diferença de que o eixo principal e o eixo cruzado são o contrário.

Como você usa e para que serve

Você pode usar uma linha simplesmente da seguinte maneira:

​  Coluna(
​  filhos: <Widget> [
​  Texto ( "Este será o lado esquerdo" ),
​  Texto ( "Este será o lado direito" )
​  ],
​  )

As opções são as mesmas que a Coluna ‘s (com o eixo transversal tornando-se a vertical eo eixo principal tornando-se a horizontal) e você pode encontrá-los no layout Opções de Widget .

Onde você usará a linha com outros widgets

Row s e Coluna s fazem para grandes widgets de ligação: com eles você pode fazer seus widgets complexos feita de elementos simples.

Por exemplo, uma postagem de rede social simples pode ser expressa com uma coluna com três elementos: uma linha mostrando a imagem do perfil e o nome do perfil, um grande texto no meio e, na parte inferior, uma linha mostrando os comandos:

​  Coluna(
​  filhos: <Widget> [
​  Fileira(
​  filhos: <Widget> [
​  ProfilePicture (profileID),
​  Texto (profileName),
​  ],
​  ),
​  Texto (postText),
​  Fileira(
​  filhos: <Widget> [
​  LikeButton (),
​  CommentButton (),
​  ShareButton ()
​  ],
​  ),
​  ],
​  )

Esta combinação de widgets, mas também apenas uma Row de widgets, é facilmente inserida dentro de um ListView ou GridView que, para manter o exemplo em execução de uma rede social, poderia ser um feed de rede social.

Isso não é difícil de fazer - começaremos respeitando os espaços e tamanhos necessários para os diferentes elementos que o compõem:

​  Coluna(
​  filhos: <Widget> [
​  Fileira(
​  filhos: <Widget> [
​  CircleAvatar (
​  backgroundImage: ProfilePicture (profileID),
​  ),
​ 
​  Preenchimento(
​  preenchimento: EdgeInsets.all (20),
​  filho: Texto (nome do perfil),
​  ),
​  ],
​  ),
​  Alinhar(
​  alinhamento: FractionalOffset.centerLeft,
​  filho: Preenchimento (
​  preenchimento: EdgeInsets.all (5),
​  filho: Texto (postText),
​  ),
​  ),
​  Divisor(),

​  LikeButton (),

​  ],

Para fazer isso, começaremos criando uma classe para conter os dados da postagem:
