Adicionar a Linha de Exclusão

O primeiro widget Row conterá o botão usado para excluir um único caractere e o botão usado para excluir a expressão inteira.

Todos os botões no aplicativo irão chamar métodos _CalculatorHomePageState , então precisamos defini-los. Voltando à definição de layout aplicativo, os botões serão FlatButton s, o que você vai adicionar ao primeiro Row widget que você pode ver no código , inserindo, dentro dos colchetes de que Row ‘s crianças atributo, o seguinte código:

layout / calculator_part1 / lib / main.dart
​  FlatButton (
​  filho: Texto (
​  'C' ,
​  style: TextStyle (color: Colors.white),
​  ),
​  onPressionado: () {deleteAll ();},
​  cor: Colors.black54,
​  ),
​  FlatButton (
​  filho: Texto (
​  '<-' ,
​  style: TextStyle (color: Colors.white)
​  ),
​  onPressionado: () {deleteOne ();},
​  cor: Colors.black87,
​  ),

Usar este código resultará no botão da esquerda (excluir tudo) ser cinza e o da direita (excluir um caractere) ser preto.

Isso ocorre porque o número após o nome da cor (que é usado dessa maneira apenas com preto e branco) especifica o quão transparente eles são; um número baixo significa que eles são muito transparentes e quase invisíveis e um número alto é usado para tornar algo muito opaco.
